# RAG 201

> You have the foundations — now take RAG to production. This section wires everything together into a fully automated, evaluated, and GitOps-managed knowledge system.

Building on RAG 101, you'll productionise the pipeline end to end:

* **Prepare Prod** — set up the production environment and infrastructure
* **RAG Pipelines** — automate document ingestion and vector store population with Kubeflow
* **Canopy with RAG** — integrate retrieval into your AI assistant so it answers from your documents
* **RAG Evals** — measure retrieval quality with an LLM-as-judge scorer tuned for RAG semantics
* **Pipeline of Pipelines** — close the loop with a Tekton trigger that runs evals automatically on every change
